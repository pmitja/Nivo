# MVP Roadmap

## Cilj MVP

Zgraditi prvo verzijo platforme, katere jedro so spletne strani, prilagodljivi kontaktni obrazci in zanesljivo zbiranje povpraševanj. Super Admin pripravi obrazec za posamezno stranko, spletna stran pa ga uporabi kot vdelan obrazec ali prek API endpointa.

## Faza 1 — Osnova

- Next.js projekt
- Auth
- Neon Postgres povezava
- Drizzle ali Prisma
- osnovni layout
- role system
- multi-tenant guard

## Faza 2 — Super Admin

- dodajanje strank
- pregled strank
- profil stranke
- urejanje paketa
- status stranke
- AI dodatek označen kot »Pride kmalu«
- Google review link
- interni zapiski

## Faza 3 — Client Dashboard

- pregled
- povpraševanja
- stranke / CRM
- SMS log
- Google ocene
- spletna stran zahtevki
- nastavitve

## Faza 4 — Lead sistem

- admin urejevalnik kontaktnega obrazca za posamezno stranko
- embed skripta za vdelavo obrazca
- company-specific endpoint za lasten obrazec na spletni strani
- javna kontaktna forma
- shranjevanje leada
- lead v dashboardu
- statusi leadov

## Faza 5 — Google ocene

- Google review link
- gumb za pošiljanje zahteve
- SMS za oceno
- review request log

## Faza 6 — Website change requests

- stranka odda zahtevek
- komentarji
- priloge
- statusi
- Super Admin obdelava

## Faza 7 — Osnovna analitika

- št. obiskov strani, če je na voljo
- št. povpraševanj
- št. SMS-ov
- št. review zahtev
- trend zadnjih 30 dni
